# Video background banner block

Description
-----------
Drupal module to create a simple custom block Video background banner.

Based on the custom js and css to manage the behavior of the Video. 

The  Video background banner is made using the paragraph module for Drupal 8.

The HTML output of the paragraph can be completely overwritten by your theme via twig templates.
 
After the module is installed you're free to modify the structure of the paragraph by adding or removing new fields. 

Requirements
------------
Drupal 8.x

Paragraph
